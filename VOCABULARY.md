# Vocabuary (คำศัพท์)

| Vocabulary     | คำอ่าน (ไทย)         | ความหมาย |
|----------------|-----------------------|----------|
| Deadline       | เดด-ไลน์              | เส้นตาย/กำหนดส่งงาน |
| Database       | เด-ตา-เบส             | ฐานข้อมูล |
| Infrastructure | อิน-ฟรา-สตรัค-เชอร์   | โครงสร้างพื้นฐานระบบ |
| Monitoring     | มอน-นิ-เทอะ-ริง       | การติดตามสถานะระบบ |
| Accessibility  | แอค-เซส-ซิ-บิ-ลิ-ที    | การเข้าถึงสำหรับผู้ใช้ทุกกลุ่ม |
| Prototype      | โปร-โท-ไทป์           | ต้นแบบเพื่อทดสอบ |
| REST API       | เรสต์-เอ-พี-ไอ         | มาตรฐานบริการเว็บ |
