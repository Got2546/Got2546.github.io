>## Glossary 
- An object inside a process that runs program instructions. If an operating system allows it, multiple threads can run on different processors at the same time.

>## Gemini
- วัตถุภายในโปรเซส (Process) ที่ทำหน้าที่ประมวลผลชุดคำสั่งของโปรแกรม หากระบบปฏิบัติการรองรับ เธรด (Thread) หลายๆ เธรดจะสามารถทำงานบนตัวประมวลผล (Processor) คนละตัวได้ในเวลาเดียวกัน

>## ChatGpt
- เป็นวัตถุภายในโปรเซสที่ทำหน้าที่รันชุดคำสั่งของโปรแกรม และถ้าระบบปฏิบัติการรองรับ ก็สามารถให้หลาย ๆ เธรดทำงานพร้อมกันบนซีพียูต่างตัวได้

>## MySelf
- มันคือเธรดในโปรแกรม ที่ช่วยให้โปรแกรมทำงานหลายอย่างพร้อมกันได้ โดยเฉพาะถ้ามีหลายซีพียูก็ยิ่งรันได้พร้อมกันจริง ๆ นะ
