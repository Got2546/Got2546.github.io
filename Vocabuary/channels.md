# --- channels ---

![This is an alt text.](/img/Channels.png )

>## Glossary 
- Web sites that you can customize with a .CDF file to give you only the information that you want to see. When you subscribe to a channel, Internet Explorer monitors the Web sites included in the channel and downloads only the information that fits the channel's parameters.

>## Gemini
-เว็บไซต์ที่คุณสามารถปรับแต่งได้โดยใช้ไฟล์ .CDF เพื่อให้ได้รับเฉพาะข้อมูลที่คุณต้องการจะดูเท่านั้น เมื่อคุณลงทะเบียนติดตามช่องข้อมูล (Subscribe to a channel) โปรแกรม Internet Explorer จะคอยตรวจสอบเว็บไซต์ที่อยู่ในช่องข้อมูลนั้น และดาวน์โหลดเฉพาะข้อมูลที่ตรงตามเงื่อนไขที่กำหนดไว้ในช่องข้อมูลมาให้

>## ChatGpt
- เว็บไซต์ที่คุณสามารถปรับแต่งด้วยไฟล์ .CDF เพื่อให้แสดงเฉพาะข้อมูลที่คุณอยากเห็น เมื่อคุณกดติดตามช่อง (channel) แล้ว Internet Explorer จะคอยตรวจสอบเว็บไซต์ที่อยู่ในช่องนั้น และดาวน์โหลดมาเฉพาะข้อมูลที่ตรงกับเงื่อนไขที่ช่องกำหนดไว้เท่านั้น

>## MySelf
-  คือเว็บแบบตั้งค่าได้ว่าอยากดูอะไรบ้าง พอกดติดตาม IE จะคอยเช็กให้เอง แล้วโหลดมาเฉพาะข้อมูลที่เราสนใจเท่านั้น
