## --------ยืนยันตัวตนเข้าใช้งาน App Thai ID------------
จัดอยู่ในหมวด Logical / Technical Controls (การควบคุมทางตรรกะหรือเทคโนโลยี)
ซึ่งเป็นส่วนหนึ่งของ Access Control (การควบคุมการเข้าถึงระบบ) 

เหตุผลเพราะ:
การยืนยันตัวตนเพื่อเข้าใช้ App Thai ID เช่น การใส่รหัส PIN, การสแกนใบหน้า หรือการยืนยันตัวตนด้วยชีวมิติ เป็นกระบวนการที่ใช้ เทคโนโลยีสารสนเทศและซอฟต์แวร์ ในการตรวจสอบว่าผู้ใช้งานเป็นบุคคลที่ถูกต้องจริงก่อนอนุญาตให้เข้าถึงระบบหรือข้อมูลสำคัญ

การควบคุมลักษณะนี้มีวัตถุประสงค์เพื่อ ป้องกันการเข้าถึงโดยไม่ได้รับอนุญาต, ลดความเสี่ยงจากการแอบอ้างตัวตน และช่วยรักษาความมั่นคงปลอดภัยของข้อมูลส่วนบุคคลภายในแอปพลิเคชัน

ดังนั้น การยืนยันตัวตนใน App Thai ID ไม่ใช่ Physical Controls (เพราะไม่ได้เป็นอุปกรณ์หรือสิ่งกีดขวางทางกายภาพ) และ ไม่ใช่ Administrative Controls (เพราะไม่ได้เป็นนโยบายหรือระเบียบข้อบังคับ) แต่จัดอยู่ใน Logical / Technical Controls อย่างชัดเจน
